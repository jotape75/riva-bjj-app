Biometria Necessária

Este app exige biometria (Face ID / Touch ID / impressão digital) para proteger o acesso.

Seu dispositivo ou navegador não suporta essa funcionalidade. Requisitos:

  • iPhone com iOS 16 ou superior (todos os browsers usam Safari/WebKit no iOS)
  • Android com biometria ativada e Chrome atualizado
  • Acesso via HTTPS (GitHub Pages já usa HTTPS ✓)

Se o seu dispositivo é compatível, tente recarregar a página ou abrir no Safari.
